RAM & ROM

RAM is a type of volatile memory. RAM uses transistors to store data, and these need electricity to work. Every time you shut down your computer, anything stored on this memory disappears. In other words: RAM is easily added to or changed.

ROM is a non-volatile memory type. This means it receives data and permanently writes it on a chip, and it lasts even after you turn off your computer.


WHAT DOES MEAN BY VOLATILE MEMORY?

Volatile memory is the memory that can keep the information only during the time it is powered up. In other words, volatile memory requires power to maintain the information.

The main difference between the two is that volatile memory requires power to maintain its data, while non-volatile memory does not. This means that if power is lost, all data stored in volatile memory will be lost, while non volatile memory will retain its data.


MOTHERBOARD

A computer’s motherboard is typically the largest printed circuit board in a machine’s chassis. It distributes electricity and facilitates communication between and to the CPU, RAM , and any other component of the computer’s hardware. There is a broad range of motherboards, each of which is intended to be compatible with a specific model and size of the computer.
